Summary: Latitude vs. Weather Data

The purpose of this project was to analyze how weather changes in relationship to latitude. Data from 500+ cities was pulled from the OpenWeatherMap API (https://openweathermap.org/api) for this analysis.


Matplotlib was used to plot various aspects of weather vs. latitude. Aspects included temperature, cloudines, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
